Lógica do projeto

Como o projeto foi programado

O projeto utiliza HTML para organizar o conteúdo, CSS para criar o visual e JavaScript para controlar o jogo, o quiz, a pontuação, o tempo e a saúde da lavoura.

  • Variáveis para controlar pontuação, tempo e saúde da lavoura.
  • Funções para iniciar, reiniciar e finalizar o jogo.
  • Eventos de clique, teclado, mouse e toque na tela.
  • Canvas para desenhar o cenário, o jogador e os itens.
  • Colisão entre o carrinho ecológico e os itens que caem.
  • Quiz interativo com verificação de respostas.
  • Manipulação do DOM para atualizar textos e pontuações na tela.

Como funciona

Objetivo do jogo

✅ Colete

Recolha itens positivos como reciclagem, água limpa, mudas e polinizadores.

⚠️ Evite

Fuja de queimadas, veneno, produtos químicos e poluição que prejudicam a lavoura.

🏆 Vença

Faça a maior pontuação possível antes que o tempo acabe e mantenha a saúde da lavoura.

Mensagem do projeto

Por que o EcoLavoura foi criado?

Com o EcoLavoura, eu quis mostrar que o campo pode ser produtivo e sustentável ao mesmo tempo. Cuidar da água, preservar o solo, proteger as abelhas e evitar a poluição são atitudes importantes para garantir alimento, vida e futuro.
